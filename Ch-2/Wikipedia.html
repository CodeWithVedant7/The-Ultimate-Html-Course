HTML: Hypertext Markup Language

Introduction

HTML (Hypertext Markup Language) is the standard markup language used to create web pages. It provides the structure of a webpage, allowing developers to format text, embed images, create links, and much more.

History

HTML was created by Tim Berners-Lee in 1991. The first version of HTML contained 18 tags and was designed for simple document formatting. Over the years, HTML has evolved significantly, with the latest version being HTML5.
